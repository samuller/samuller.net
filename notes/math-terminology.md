$\ a + b = c$

- Terms: $a$ and $b$
- Sum: $c$
- Operator: $+$

$\ a - b = c$

- Difference: $c$

$\ a \times b = a\cdot b = ab = c$

- Factors: $a$ and $b$
- Product: $c$

$\ a \div b = a / b =  \frac{a}{b} = c$ remainder $d$

- Dividend: $a$
- Divisor: $b$
- Quotient: $c$
- Remainder: $d$

$\ \frac{1}{a} = b$

- Reciprocal of $a$: $b$

$\ a^b = c$

- Base: $a$
- Exponent/power: $b$
- Product: $c$
- $b$-th root of $c$: $a$

$\ \log_ab$

- Base: $a$

$\ ax^2 + bx + c = y$

- Constants: $a$, $b$, $c$
- Variables: $x$, $y$
- Coefficients: $a$, $b$

$\ f(x) = y$

- Function: $f$
- Input: $x$
- Output: $y$
- Inverse of function $f$: $f^{-1}(x)$
- Derivative of $f$ ("f prime of x"): $f^{\prime}(x)$
- Alternative notations for derivative: $f'(x) = \frac{\mathrm{d}f}{\mathrm{d}x} = \frac{\mathrm{d}y}{\mathrm{d}x} = \dot{y}$
